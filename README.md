# Jared Poulsen — UX Design Portfolio

A UX design portfolio with homepage, project pages, and about page. Project pages feature a structured case study format with CHAOS (Audit, Research, Explore) and ORDER (Systemize, Implement, Refine) sections.

## Structure

- **Homepage** — Hero, tagline, and featured work
- **Projects** — List of passion projects
- **Project Page** — Hook section, project info, CHAOS/ORDER sections with horizontally scrollable subsections and cards
- **About** — Bio and background

## Project Page Features

- **Hook section** — Problem statement and narrative
- **Project info** — Case study details, role, team, objective
- **Section nav** — Jump links to CHAOS and ORDER subsections
- **Horizontal scroll** — Subsections scroll horizontally with chevron buttons (right-aligned)
- **Process cards** — Header, text, image; magnifying glass opens lightbox with enlarged view + extra content
